Transformation and site videos

1. Export one file as MP4 (H.264 + AAC is widely supported on phones).
2. Name the first garden before/after reel exactly:

   garden-transformation-01.mp4

3. Place it in this folder (same level as this README).

The home page checks for that file; when it is present after deploy, the video player appears automatically.

Optional second file later: add garden-transformation-02.mp4 and wire it in src/App.jsx.
