--------------------------------------------------------------------------------
FIX: YOU PASTED THE SCRIPT TWICE IN GOOGLE
--------------------------------------------------------------------------------
1. Open the sheet → Extensions → Apps Script.
2. Open Code.gs. Press Ctrl+A (select all) → Delete.
3. Open FOR-GOOGLE-APPS-SCRIPT-PASTE-ONLY.txt on your PC → copy ONLY the JavaScript (between the “copy” lines) → paste once into Code.gs.
4. Set var SECRET = '...' and optional DRIVE_FOLDER_ID.
5. Save → Deploy → Manage deployments → Edit → New version → Deploy.

You must have exactly ONE of each: doPost, doGet, outJson, saveQuotePhotos_. Duplicate function names break the project.
