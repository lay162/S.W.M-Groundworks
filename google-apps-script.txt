Quick files in this folder:
- FOR-GOOGLE-APPS-SCRIPT-PASTE-ONLY.txt → paste ONLY that into Apps Script Code.gs (avoids pasting the guide twice).
- GITHUB-PAGES-HOSTING.txt → GitHub Pages + custom domain + VITE_* secrets for the quote form.

--------------------------------------------------------------------------------
FIX: YOU PASTED THE SCRIPT TWICE IN GOOGLE
--------------------------------------------------------------------------------
1. Open the sheet → Extensions → Apps Script.
2. Open Code.gs. Press Ctrl+A (select all) → Delete.
3. Open FOR-GOOGLE-APPS-SCRIPT-PASTE-ONLY.txt on your PC → copy ONLY the JavaScript (between the “copy” lines) → paste once into Code.gs.
4. Set var SECRET = '...' and optional DRIVE_FOLDER_ID.
5. Save → Deploy → Manage deployments → Edit → New version → Deploy.

You must have exactly ONE of each: doPost, doGet, outJson, saveQuotePhotos_. Duplicate function names break the project.

--------------------------------------------------------------------------------
HEADERS ROW 1 (A1:K1)
--------------------------------------------------------------------------------
Timestamp | Name | Email | Phone | Service | Message | IP | PhotoCount | PhotoNames | PhotoLinks | Marketing consent

Add column **K** (Marketing consent) if your sheet was created from the old template — values **Yes** / **No** from the quote form checkbox.

--------------------------------------------------------------------------------
THE “SECRET” (GOOGLE_APPS_SCRIPT_SECRET) — WHAT IT IS
--------------------------------------------------------------------------------
It is NOT a Google password and you do not “look it up” anywhere.

1. Invent one long random phrase only you know, e.g. swmQuoteLog-k9mP2xQ7vL4nR8
2. In Apps Script (Code.gs), first line inside doPost:  var SECRET = 'paste-the-same-phrase-here';
3. In GitHub → repository → Settings → Secrets and variables → Actions → add **Repository secrets**:
     VITE_GOOGLE_APPS_SCRIPT_SECRET   =   the exact same phrase (no spaces at start/end)
     VITE_GOOGLE_APPS_SCRIPT_URL      =   your Web App /exec URL
4. Run the **Deploy GitHub Pages** workflow (or push to main) so the site rebuilds with those values.

The website’s server sends that phrase inside the JSON body; the script checks data.secret === SECRET so random people cannot spam your sheet.

--------------------------------------------------------------------------------
GITHUB ACTIONS (BUILD-TIME VITE_* VARS)
--------------------------------------------------------------------------------
VITE_GOOGLE_APPS_SCRIPT_URL, VITE_GOOGLE_APPS_SCRIPT_SECRET (match SECRET in the script). Optional:
VITE_GOOGLE_APPS_SCRIPT_INCLUDE_PHOTOS=true for Drive uploads (large payloads may fail from the browser).

Sheets only (no Drive): leave DRIVE_FOLDER_ID = '' in the script; do NOT set INCLUDE_PHOTOS (or set false). Rows still get PhotoCount + PhotoNames.

There is no server email path in this repo anymore; rely on the Sheet and/or MailApp inside Apps Script if you add it.
